# System Monitor Dashboard

this is just tempalte for my future projects 

A retro-styled analytics dashboard using Node.js and SQLite. 

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   node backend/server.js

3. Open in browser:
   http://localhost:3000
